# MiniTeremimArduino
Teremim feito com um buzzer e um sensor ultrassonico de distância

# Materiais
Para montar esse projeto foram usados os seguintes materiais:

* Arduino Uno
* Buzzer
* Sensor de distância (HC-SR04)
* Alguns jumpers
* Marcas de distância (no caso a cada 4cm)
